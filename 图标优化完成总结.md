## ✨ 动画效果升级

### 🎬 **9 种专属动画**

1. **sparkle（闪烁）** - 生成按钮悬停
   - 光晕呼吸效果
   - 1.5s 无限循环

2. **celebrate（庆祝）** - 生成成功
   - 左右摇摆 ±15°
   - 0.6s 一次

3. **check-pop（复制确认）** - 复制成功

   - 0.5s 一次

4. **shake-warning（警告摇摆）** - 删除悬停
   - 左右晃动 ±3°
   - 0.4s 一次

5. **trash-shake（垃圾桶晃动）** - 删除点击
   - 更大幅度 ±8°
   - 0.5s 一次

6. **sweep-motion（扫帚挥动）** - 清空全部

   - 0.6s 一次

7. **arrow-slide（箭头滑动）** - 分页按钮
   - 左/右滑动 4px
   - 0.6s 无限循环

8. **icon-bounce（弹跳）** - 所有按钮点击
   - 向上 4px 再回落
   - 0.3s 一次

9. **icon-pulse-glow（脉冲光晕）** - 吸引注意
   - 放大 1.08x + 光晕

## 🔧 技术实现

### CSS 选择器策略
```css
/* 通用 SVG 样式 */
.btn svg {
  transition: all 0.3s cubic-bezier(0.4, 0, 0.2, 1);
}

/* 特定按钮样式 */
.generate-btn:hover:not(:disabled) svg { ... }
.copy-icon-btn:hover:not(:disabled) svg { ... }
.delete-msg-btn:hover:not(:disabled) svg { ... }
```

### 动画性能优化
```css
/* 使用 transform 和 opacity（GPU 加速） */
transform: scale(1.15) rotate(15deg);
filter: drop-shadow(...);  /* 代替 box-shadow */
```

### 响应式适配
```css
/* 触摸设备优化 */
@media (hover: none) {
  .btn:hover svg {
    /* 禁用悬停动画 */
  }
}
```

## 🎯 核心优势

### 1. **统一视觉语言**
所有图标采用相同的设计风格：
- 2px 描边粗细
- 圆角端点
- 线性设计
- 无填充

### 2. **丰富的互动反馈**
每个按钮都有 3-5 种状态：
- 默认状态
- 悬停状态（放大/旋转/平移）
- 点击状态（弹跳/缩小）
- 成功状态（绿色脉冲）
- 失败状态（红色脉冲）

### 3. **语义化设计**
图标准确传达功能：
- ✅ 加号圆圈 = 创建/生成
- ✅ 垃圾桶 = 删除
- ✅ 复制文件 = 复制
- ✅ 循环箭头 = 刷新
- ✅ 禁止圆圈 = 清空/取消

### 4. **无障碍支持**
每个 SVG 都包含：
```vue
<svg 
  role="img" 
  aria-hidden="true"
  focusable="false"
>
```

## 💡 使用建议

### 1. 保持一致性
所有新按钮都应使用相同风格的 SVG：
```vue
<svg 
  viewBox="0 0 24 24" 
  width="18" 
  height="18" 
  fill="none" 
  stroke="currentColor" 
  stroke-width="2" 
  stroke-linecap="round" 
  stroke-linejoin="round"
>
  <!-- 路径 -->
</svg>
```

### 2. 合理使用动画
- ✅ 悬停：轻微动画（吸引注意）
- ✅ 点击：快速反馈（0.3-0.5s）
- ✅ 成功/失败：明显反馈（0.5-0.9s）
- ❌ 避免：过长动画（>1s）、过多同时动画

### 3. 图标尺寸指南
- 大按钮：20px
- 标准按钮：18px
- 小按钮：16px
- 内联图标：14px
